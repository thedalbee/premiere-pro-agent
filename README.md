# premiere-pro-agent

> CLI that lets AI agents edit your Adobe Premiere Pro timeline.

`ppro` — one command to cut silence, build rough cuts, and place captions in Premiere Pro, driven by any AI agent (Claude Code, Codex, Cursor) or your own scripts.

**Status: name reserved — first release coming soon.**

- 100% local: localhost-only bridge, no telemetry, nothing leaves your machine
